# 📦 Amazon Sales Dashboard

An interactive Excel dashboard built from Amazon sales data to track revenue, 
top products, and regional performance.

## Tools Used
- Pivot Tables & Pivot Charts
- SUMIFS / COUNTIFS
- Slicers (interactive filtering)
- Conditional Formatting

## What It Shows
- Monthly revenue trends
- Top-selling products
- Sales by region
- Revenue by category

## Key Insights
- [Example: Electronics made up 35% of revenue from only 15% of orders]
- [Example: Highest sales came from (your top region)]
- [Example: Sales peaked in (your top month)]



## How to Use
1. Download the `.xlsx` file from the `dashboard` folder
2. Open in Excel
3. Click slicers to filter by category/region/date
